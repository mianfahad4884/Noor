# **App Name**: Veridia Guide

## Core Features:

- Dashboard & Daily Insights: Displays a personalized greeting, current Hijri date, next prayer time countdown, and daily rotating Ayah and Hadith fetched via external APIs, adaptable for web viewing.
- Interactive Tasbeeh Counter: A digital counter with customizable dhikr phrases and target counts, including session history saved to client-side storage, with visual feedback for web taps.
- Comprehensive Dua Library: A searchable and categorized collection of Duas, displaying Arabic text, transliteration, and translation, with a user bookmarking system managed client-side.
- AI Islamic Text Explainer Tool: A generative AI tool that provides contextual explanations, deeper insights, and historical background for provided Quranic verses or Hadiths.
- Qibla Direction Finder: An interactive Qibla finder leveraging browser geolocation and compass capabilities (if available) to accurately determine the prayer direction for web users.
- Personalized Settings & Profile: Allows users to manage app preferences including appearance (dark/light mode), language, content sources, and configure web browser notification reminders for prayers or daily content.

## Style Guidelines:

- Primary accent color: A deep, spiritual green (#1B4332) selected for its association with tranquility and nature, defining key interactive elements and branding accents.
- Warm accent color: A rich gold (#C9A84C) to highlight important actions, spiritual indicators, and provide an elegant contrast, lending a sense of tradition and value.
- Light background scheme: A soft, warm parchment (#F7F4EF) as the primary background, with elevated UI surfaces presented in a subtle, warm-tinted white (#FFFFFF) to maintain a premium feel.
- Dark background scheme: A deep forest night (#0D1F17) forming the dark mode's primary background, complemented by slightly lighter dark green surfaces (#152B1E) for depth and readability.
- Arabic text will use 'Amiri' (serif) for its beautiful and authentic representation. Note: currently only Google Fonts are supported.
- Headings will utilize 'Lora' (serif) to convey an elegant, intellectual, and contemplative tone throughout the application. Note: currently only Google Fonts are supported.
- UI elements and body text will be set in 'Inter' (sans-serif) for optimal legibility, modern aesthetic, and versatility across various screen sizes. Note: currently only Google Fonts are supported.
- A meticulously designed custom SVG icon set will be used across the application, prioritizing elegance, clarity, and thematic relevance, avoiding generic iconography.
- The application will feature a prominent hero section on the homepage, a clean two-column grid system for categorized content, and a clear, persistent primary navigation accessible throughout the site.
- Subtle and graceful micro-animations, such as soft fade and slide-up effects for element entry, slight scaling for button presses, and gentle pulse animations for counter interactions, will enhance the premium and responsive user experience.
